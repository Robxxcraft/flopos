## Frontend Flopos

Frontend Flopos
-Vue 3.3
-Tailwind 3
-Swiper.js 10
-Chart.js & vue chart 3

### 1. Masuk ke dalam project yang telah disalin

```sh
cd front-flopos
```
<br />


### 2. Install package yang dibutuhkan

```sh
npm install
```
<br />


### 3. Jalankan project aplikasi

```sh
npm run dev
```
